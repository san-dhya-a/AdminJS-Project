# FAQ Module

This module manages the Frequently Asked Questions (FAQ) section of the application.

## Features
- **Title & Subtitle**: Main headings for the FAQ page.
- **Description**: General context or introduction text.
- **Dynamic FAQ Sections**: Add multiple FAQ entries (Question/Answer style) using the dynamic sections tool in AdminJS.

## Data Structure
Each FAQ entry in the dynamic list contains:
- `title`: The question.
- `subtitle`: Optional sub-text or category.
- `description`: The answer content.

## AdminJS Access
Accessible under the **FAQ** menu in the sidebar.
