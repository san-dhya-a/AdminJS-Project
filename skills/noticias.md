# Noticias Module

This module handles the application news feed, articles, and blog-like content.

## Features
- **Title**: Headline of the noticia.
- **Description**: Main article content.
- **Category**: Managed through a selection field.
- **Date**: Auto-generated publication date.
- **Images**: Supported through AdminJS asset uploads (if configured).

## Usage
Used for announcements, internal communications, and application-wide notifications.
