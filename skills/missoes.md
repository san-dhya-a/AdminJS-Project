# Missoes Module

This module manages the interactive missions and tasks for the application's users.

## Features
- **Title**: Mission name (e.g., "Complete Profile").
- **Description**: Instructional or motivational text.
- **Points/Rewards**: Configuration for value assigned to the mission.
- **Image/Icon**: Symbol or background image selection.
- **Expiry Date**: Manage when missions go off-line.

## Progress Configuration
Define mission states:
- Active
- Completed
- Expired
- Archived
