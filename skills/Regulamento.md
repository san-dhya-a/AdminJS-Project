# Regulamento Module

This module manages the official regulations and terms for the application.

## Features
- **Title**: Section name (e.g., "General Terms").
- **Description**: Main introductory or summary text.
- **Dynamic Entries**: Add multiple regulation items with detailed clauses and conditions.

## Data Structure
Each regulation entry contains:
- `title`: The clause or rule title.
- `description`: The detailed legal or regulatory text.

## AdminJS Access
Accessible under the **Regulamento** menu in the sidebar.
